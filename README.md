# 酒店后台管理系统 java877

## 项目概述

酒店后台管理系统是一套基于 Spring Boot、Java 和 MySQL 技术栈，前端使用 Vue.js 构建的现代化管理系统。它为酒店行业提供了一套全面、高效的功能模块，旨在优化酒店运营流程，提高管理效率。

## 技术栈

- 后端：Spring Boot、Java
- 数据库：MySQL
- 前端：Vue.js

- JDK 1.8 或更高版本
